# Natac - Online Catan Replica

A multiplayer online version of the classic board game Catan built with Node.js and Socket.IO.

## Quick Start

```bash
npm install
npm start
```

Open http://localhost:3000 to play!

## Features

- Real-time multiplayer gameplay
- Web-based interface
- Simple deployment

## Tech Stack

- Node.js
- Express
- Socket.IO
- HTML/CSS/JavaScript
